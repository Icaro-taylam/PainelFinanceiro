# 💰 FinanceFlow Pro

![Licença](https://img.shields.io/badge/license-MIT-green)
![Versão](https://img.shields.io/badge/version-1.0.0-blue)
![Responsivo](https://img.shields.io/badge/mobile--friendly-YES-brightgreen)

**FinanceFlow Pro** é um ecossistema financeiro pessoal leve, rápido e totalmente responsivo. Projetado para funcionar perfeitamente em dispositivos móveis e desktop, o sistema permite o controle total de entradas e saídas com visualização de dados em tempo real.

---

## 🚀 Funcionalidades

-   **Dashboard Financeiro:** Controle de receitas e despesas (água, luz, gás, salário, etc).
-   **Gráfico Interativo:** Visualização em pizza (Doughnut Chart) usando a biblioteca Chart.js.
-   **Calculadora Integrada:** Calculadora nativa customizada para cálculos rápidos sem sair da página.
-   **Bloco de Notas:** Espaço para anotações e planejamentos com salvamento automático.
-   **Sistema de Temas:** Alternância entre Modo Claro (Light) e Modo Escuro (Black/Dark).
-   **Persistência de Dados:** Todos os dados são salvos no `LocalStorage` do navegador (não perde os dados ao fechar a aba).
-   **Exclusão Inteligente:** Opção para remover registros lançados incorretamente.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando o conceito de **Vanilla JS** (JavaScript puro), garantindo máxima performance:

* **HTML5:** Estruturação semântica.
* **CSS3:** Design responsivo com Flexbox, Grid e Variáveis de Cores (Root).
* **JavaScript (ES6+):** Lógica de cálculos, manipulação de DOM e LocalStorage.
* **Chart.js:** Biblioteca para geração de gráficos profissionais.
